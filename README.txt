HƯỚNG DẪN CÀI ĐẶT & SỬ DỤNG PROJECT WEBSITE BÁN QUẦN ÁO ONLINE

1. Yêu cầu:
- Cài XAMPP bản 3.3.0 hoặc tương đương (https://www.apachefriends.org/)
- PHP >= 7.4, MySQL >= 5.x

2. Cài đặt:
- Copy folder "MeomeoFashion" vào thư mục htdocs của XAMPP
- Mở XAMPP, khởi động Apache & MySQL
- Lấy file fashionofficial.sql trong folder 
- Import file sql trong phpMyAdmin (http://localhost/phpmyadmin)

3. Sử dụng:
- Truy cập web: http://localhost/MeomeoFashion/pages/index.php
- Đăng ký tài khoản, đăng nhập: email: thanhha@gmail.com, mật khẩu: meomeo
